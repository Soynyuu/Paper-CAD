# 未踏ジュニア 2025年度プロジェクト

このディレクトリには2025年度未踏ジュニアプログラムの成果物が含まれています。

## プロジェクト

### 1. chili3d
Web-based 3D CAD application for online model design and editing

- **技術スタック**: TypeScript, Three.js, OpenCascade (WebAssembly)
- **リポジトリ**: https://github.com/Soynyuu/chili3d
- **開発**: `cd chili3d && npm install && npm run dev`

### 2. unfold-step2svg
3D-to-2D papercraft generation system

- **技術スタック**: Python, FastAPI, OpenCASCADE (pythonocc-core)
- **リポジトリ**: https://github.com/Soynyuu/unfold-step2svg
- **開発**: `cd unfold-step2svg && conda env create -f environment.yml && conda activate unfold-step2svg && python main.py`
